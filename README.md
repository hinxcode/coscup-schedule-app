# COSCUP 議程表

以 ReactXP 開發的 React Native App (iOS, Android) 及 Web

### Setup

- `npm install`

### Building for Web

- `npm run web-watch`
- 以瀏覽器打開 `index.html`

### Building for React Native

- 方式 1:
	- `npm run ios` 及 `npm run android`
- 方式 2
	- `npm start` 執行 React Native Packager
	- 使用 Xcode 及 Android Studio 編譯
